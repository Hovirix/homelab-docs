## Servers

### Beelink S12 Pro (Active)

The main server running all homelab services.

| Spec | Details |
|------|---------|
| CPU | Intel N100 (4 cores, 3.4GHz) |
| RAM | 16GB DDR4 |
| Storage | 500GB NVMe + 2TB SSD |
| OS | Fedora CoreOS |
| Role | Primary server (Podman containers) |

<Tip>
The N100 is power-efficient (~6W idle) and handles containerized workloads well. Perfect for a 24/7 homelab.
</Tip>
