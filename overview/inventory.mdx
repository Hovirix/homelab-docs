## Overview

All services run as Podman containers on Fedora CoreOS, managed via Quadlet (systemd units).

## Deployment

All containers are defined as Podman Quadlets in `/etc/containers/systemd/`.

```bash
# List running containers
podman ps

# Check service status
systemctl status <service>.container

# View logs
journalctl -u <service>.container
```

<Tip>
Quadlet files are stored in a Git repo and deployed via SCP. See [Architecture](/overview/architecture) for the full deployment workflow.
</Tip>

## Adding New Services

1. Create a `.container` Quadlet file
2. Add to Git repo
3. Deploy via SCP to `/etc/containers/systemd/`
4. Run `systemctl daemon-reload && systemctl start <service>.container`
5. Configure Caddy route
6. (Optional) Set up Authentik application for SSO
